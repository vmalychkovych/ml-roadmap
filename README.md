# ML Roadmap

## Setup

This project works with `Python 3.12`.

### Create and activate a virtual environment

PowerShell:

```powershell
py -3.12 -m venv .venv
.\.venv\Scripts\Activate.ps1
python -m pip install -r requirements.txt
```

### Run the TensorFlow example

```powershell
python .\week2\mnist_nn.py
```

If `python --version` points to `3.14`, use `py -3.12` instead.
